# Voxa API

This is API for Voxa (social media platform). 
See for API Spec at [Voxa API Docs](https://voxa-api-docs.vercel.app/).
Built with Java and Spring Boot.
Still on progress and feature will be added along the way.

## Tech Stack

- Java 21
- Spring Boot 4
- Spring Data JPA
- Spring MVC
- Spring Security
- MySQL
- Flyway
- JWT
- Maven
- Lombok
- JUnit & Mockito
- Bean Validation
